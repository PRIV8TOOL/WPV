## How To Install

**Type this command to your terminal/putty software:**

1) _`apt-get update`_
2) _`apt-get install -y git-core`_
3) _`git clone https://github.com/W3LLLL/WPV.git`_
4) _`cd WPV`_
5) _`chmod 777 install.sh`_
6) _`bash install.sh`_

**Note :** 
1) _Whenever you logout the ROOT from putty remember to put command_ `cd WPV` _when you relogin._
2) _Command to run is_ `./WPV`
